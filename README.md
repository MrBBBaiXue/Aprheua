# Aprheua

A GUI Interface for OpenCV Sharp HAAR Classifier, with Opera Classifier Data built-in.

# Build from Source

You need **Visual Studio 2019** with **.NET Framework 4.7.2** installed.

Clone this repository, open **Aprheua.sln** and hit build.
